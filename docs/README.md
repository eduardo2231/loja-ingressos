## Description

A sleek and intuitive ticket sales management system built with Python and Tkinter. Manage events, add tickets, track prices and quantities, and simulate purchases easily.

---

## Features

* Add, remove, and manage tickets.
* View ticket details directly from the app.
* SQLite database integration for persistent storage.
* Simple GUI interface built with Tkinter.

---

## Installation

1. Clone the repository:

```
git clone https://github.com/eduardo2231/Sistema-de-Vendas-de-ingressos.git
```

2. Navigate to the project folder:

```
cd Sistema-de-Vendas-de-ingressos/src
```

3. Install dependencies (if any):

```
pip install -r requirements.txt
```

4. Run the app:

```
python ticket_app.py
```

---

## Usage

* Open the app to view available tickets.
* Use the "Conta" button to add or remove tickets.
* Click on a ticket to view details or simulate a purchase.
